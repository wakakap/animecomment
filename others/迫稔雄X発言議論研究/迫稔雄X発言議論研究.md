# 迫稔雄X発言議論研究

## 背景 
[迫稔雄](https://ja.wikipedia.org/wiki/%E5%98%98%E5%96%B0%E3%81%84)是一位日本漫画家，2005年作品「嘘喰い」获奖并连载在『週刊ヤングジャンプ』，截至2022年，该系列发行超过1000万部。并被真人电影改编。其作品在[bangumi](https://bangumi.tv/subject/67104)上8.7分的高分，书籍类排名第83。

## 问题和前提
前提：
- 有日本和香港的授权出版，但没有英文版

问题：
- 在该前提下，阅读英文翻译的盗版漫画是否允许？
分问题：
- 法律上。。。是否允许？
- 道德上。。。是否允许？
  - 绝对道德观
  - 相对道德观
- 普世价值观上。。。是否允许？

## 我的解析

- ？看盗版行为本身是错的，无耻的，没有任何辩解余地，任何理由都是借口。
   - X 如果我不看盗版，我怎么能看到这部作品？
     - √ 作者没有义务给所有人提供渠道。
   - X 盗版为作品合作者提供了人气，客观有助于作品销量。 
     - √ 无关论证，和道德无关。
   - X 现在世界上的游戏流行提供免费再收费的营销手段，这反而带来更大的利益。
     - √ 无关论证，和道德无关。另外，怎么制定营销手段是作者的权力，即使他不在乎利益最大化，也不是道德指摘的问题。
- 无知看盗版的情况
  - 不改变盗版行为对作者的影响。但改变盗版使用者是否是故意损害作者权益的判断。
  - 对于无知地看盗版，但是现在知晓了盗版不对的观众，他们该怎么办？作者是否过于冷酷。
    - 作者没有对这些人表示同情，这似乎是有点不尽人情的，但似乎够不上道德问题。
- 贫穷的情况
  - 同上
  - 假设知识教材、学术文章的盗版是道德上允许的，理由是用于个人研究或知识获取，漫画是否通用？
    - 漫画是「嗜好品」，和知识类书籍有区别。
- 作者本人骂盗版用户「バカ」，说「不在乎讲英文的市场」等言论让喜欢作品的英文盗版使用者伤心，是否道德？
  - 比起阅读盗版要好很多，并在反击的语境下可以理解。



- 我认为这场大论战体现了日本传统中一板一眼的思维。盗版是偷东西，是非常显而易见的罪行。自己没有能力看漫画，用非法途径去享受，还给自己找各种借口，简直无法理解。从这个角度我真觉得日本人的逻辑思维能力超越世界。
- 但反面上，道德到底是什么。是让大家平等的基本规范，还是维护强者地位的盾牌。从这个角度，盗版是弱者自救的通道，但是问题就在于这个漫画根本谈不上知识，不会为贫穷的人带来什么提升以改变生活质量。就好比一个穷人快饿死了偷了一个面包可以理解，但这个穷人去妓院嫖娼不给钱就不能理解了。这样看来，作者仿佛「强者」的冷酷姿态是可以理解的。他只是在咒骂嫖娼不给钱的穷人，而不是咒骂偷面包的穷人。
- 这个引申的话题很有启发性，使用盗版，能提高生活质量，但使用盗版干什么，是选择可以改变人生的知识技能还是一时享受的娱乐，会带来截然不同的结果。穷人们应当醒悟这一点。
